Abstract

In quantitative finance, particularly in optimized trade execution, reinforcement learning (RL) has shown promise due to its ability to interact with market environments based on real data. However, traditional RL methods suffer from slow research speed and rely on static market assumptions which not consider the agent execution action impact to the environment, which are inadequate for dynamic environments. To address these challenges, we propose a Self-Evolutional Single-Agent Multi-Agent RL (SE-RL) framework. This framework leverages large language models (LLMs) to designed and improve the different RL algorithm module including agent model design, state space, reward function, profiling, communication and state imagination through LLM generate module output or generate code ways. It includes a dual-enhancement mechanism at both high-level (prompt refinement and reflection) and low-level (weight fine-tuning) to improve the accuracy of LLM-generated RL algorithms. Additionally, we use a multi-agent system to simulate dynamic financial markets, accounting for the impact of order executions on market dynamics. To further enhance training with such dynamic market, we propose a machine semi-unlearning method that alternates between static and dynamic environments, integrating real historical market data and simulated market impacts. Comprehensive experiments on 200 realistic stock datasets demonstrate that our proposed framework outperforms current state-of-the-art baselines, offering a more efficient and practical solution for trade execution in dynamic markets.
